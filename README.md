# Unity3d_Robot
在Unity3D环境下研究机器人的运动学，包括运动学正反解，轨迹规划，插补等

# 1.环境

Unity3D版本: Unity3D 2018.3.8f1；

Vs版本:VS2017

# 2.功能

1.机器人运动学正解、反解，一共八组解

2.简单的直线插补

3.直线运动，点到点运动

# 3.常用函数示例

1.CMove 以点对点的方式运动到目标位置，轨迹不确定

![PtP](/png/PtP.gif)

2.CLine 以直线的方式运动到目标位置，轨迹为直线

![Line](/png/Line.gif)

3.写字函数，给定点坐标画出轨迹

![Write](/png/Write.gif)

4.构型切换,切换当前位姿的其他构型，也就是其他的解

![CC](/png/ChoosePose.gif)
